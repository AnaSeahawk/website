# Living Archive Thread Vocabulary

Threads are broad cross-time lines used by generated archive views.

Use them lightly. A record should usually carry one to three threads. More
specific detail belongs in `entry_points`, `inquiries`, `phase`, `summary`, or
the body of the source record.

## Current Thread List

| Thread | Use for |
|---|---|
| `clearing-nourishment` | Diet, simplification, salt/chloride transition, clearing, nourishment, and the body's changing baseline. |
| `community-transmission` | Consent-held community movement, practitioner stories, calls, shared observation, and relational transmission. |
| `current-state` | Current-state syntheses and milestone reviews of where the work stands. |
| `distillation-record` | Distillation runs, fraction behavior, meter readings, redistillation, and measured run comparison. |
| `fresh-plasma` | Fresh source material, fresh baseline readings, raw variation, and pre-aging source behavior. |
| `full-cycle` | Complete return/cycling practice, household flow, full output cycling, and closed-loop practice context. |
| `lineage-recognition` | First recognition, alchemical lineage, old symbols becoming physical, and the threshold where the archive became responsibility. |
| `living-record` | Dated journal, dream, body, travel, household, and field-life records behind the archive. |
| `menstrual-artava` | Menstrual/artava material, cycle-specific observations, and cycle-material distillation questions. |
| `rasa-qi-flow` | Rasa, qì, movement, participation, fluid-body perception, and restoration of flow. |
| `restoration-of-continuity` | Tissue softness, mobility, reconnection, fluidity, long-held restriction releasing, and continuity returning. |
| `source-state` | Timing, concentration, raw source condition, daily variation, and source-material comparison before interpretation. |
| `three-waters` | Solar, Lunar, Saturn, fraction language, and the still's three-register translation. |
| `topical-practice` | Skin, topical application, external practice, partner dimension, and observed body-surface changes. |

## Retired Or Folded Labels

These labels appeared in the pilot and are folded into the current list:

| Old label | Use now |
|---|---|
| `alchemical-lineage`, `first-distillation`, `recognition` | `lineage-recognition` |
| `clearing`, `nourishment`, `sodium-chloride` | `clearing-nourishment` |
| `daily-variation`, `hydrogen`, `measurements`, `redistillation` | `distillation-record` or `source-state`, depending on the record |
| `distillation` | `distillation-record` |
| `distilled-plasma` | `restoration-of-continuity` or `community-transmission` |
| `fraction-language` | `three-waters` |
| `household-flow` | `full-cycle` |
| `movement`, `participation`, `qi`, `rasa`, `somatic-practice` | `rasa-qi-flow` |
| `sexuality`, `skin` | `topical-practice` |
| `synthesis` | `current-state` when it is a state report; otherwise use the topical or distillation thread it actually serves |

## Stewardship Rule

Add a new thread only when at least three existing or expected records need it
and none of the current threads can hold the line without confusion.
