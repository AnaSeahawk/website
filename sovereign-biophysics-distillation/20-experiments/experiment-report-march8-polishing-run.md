# Experiment Report: March 8 — The Polishing Run

**Operator:** Ana Seahawk  
**Date:** March 8, 2026  
**Location:** Torremolinos, Andalusia, Spain  
**Input:** 3-litre master batch of aged Phase II Pure Plasma — entering chamber at 993 USCM  
**Run structure:** Third distillation pass (polishing run) — tracking the threshold at which residual volatile compounds clear and the fluid stabilises as a pure carrier

---

## Run Context

The input for this run was previously distilled material (Phase II Pure Plasma) that had been aged, re-entering the chamber at 993 USCM. The goal was to track in real time the depletion of residual volatile compounds and identify the exact point at which the fluid transitions into a fully stable, low-resistance carrier.

The physical behaviour of the fluid at the start of the run confirmed the cleanliness of the input: no foam, no harsh aroma — only the light, earthy character of the dietary baseline.

## Key Observations

**The 0 mV threshold.** Reading 4 marks the point at which ORP flips from negative to slightly positive (+2 mV) and conductivity drops to 870 USCM — the observable cut where the Spirit has cleared the column and what remains is stabilised water. The cooled final fraction at 0 mV and 133 USCM represents a fully neutral electrochemical state: neither electron-donating nor electron-receiving, carrying minimal dissolved load.

**The polishing run pattern.** Third distillation of aged input does not behave like a simple third pass of fresh material. The conductivity drop at Reading 3 (mid-run, 1375 USCM) is steep — a visible threshold rather than a gradual decline. The Spirit clears in a defined band, not continuously across the run.

**Retentate stability under extended thermal extraction.** The aged input Saturn portion remained workable after the longer, deeper reduction used in this run. This contrasts with fresh-run residue behaviour and confirms that gestation is the enabling condition for retentate quality, not run duration alone.

**The 3-pass law for aged plasma.** For aged input, Phase II distillate commonly enters a third run at elevated conductivity (~993 USCM). The polishing run reduces this to the 133 USCM stable range. Fresh plasma reaches that threshold in two passes; aged plasma, which carries more dissolved conductive material even after two passes, requires a third to reach the same stable endpoint.
