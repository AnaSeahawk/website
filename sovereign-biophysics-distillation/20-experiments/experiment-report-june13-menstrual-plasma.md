# Experiment Report: June 13 — Menstrual Plasma Distillation Series

**Operator:** Ana Seahawk
**Date:** June 13, 2026
**Input:** Morning menstrual plasma collection (~300 mL), filtered through woven hemp over felted wool
**Run structure:** Raw morning collection → first distillation (blow-over) → redistillation of the blow-over batch; one later raw collection recorded as a reference sample, not distilled

*This is the first run in the archive to include menstrual blood — artava, the feminine fraction. The plasma was collected and distilled within a few hours.*

---

## Operator's Account

This run felt like a threshold before the meters said anything.

The material was visibly different because the menstrual fraction was present. It was not an abstract cycle note or a symbolic correspondence. Artava entered the jar as part of the source material, filtered through woven hemp over felted wool, then brought to the still while still fresh from the morning. The feminine fraction was no longer only a theoretical line in the archive. It was in the vessel.

The first reading made the difference plain: EC 10,690 µS/cm, TDS 5,400 ppm, pH 5.1. The raw morning collection was carrying a density none of the recent non-menstrual fresh runs had shown. That number changed the quality of attention in the room. The run was no longer only another fresh-plasma comparison. It had become the first measured entry of menstrual material into the distillation record.

The blow-over kept the record honest. The first distillation did not produce a clean ideal sample; it carried material over and had to be redistilled. That matters. The second pass became part of the question: what remains of the charge signature when this menstrual-fraction material is clarified again?

What held was the familiar arc. Acidic raw source to alkaline distillate. Positive ORP to negative ORP. Conductivity falling with each pass. H₂ appearing after distillation and remaining present through redistillation. The run did not answer what artava changes. It opened the cycle question in a form the archive can return to.

The body has a new line of inquiry now.

## Run 1 — Raw Morning Collection (Filtered)

Approximately 300 mL morning collection containing menstrual blood, filtered through woven hemp over felted wool before distillation.

| Parameter   |      Reading |
| :---------- | -----------: |
| Temperature |       25.2°C |
| pH          |          5.1 |
| EC          | 10,690 µS/cm |
| TDS         |    5,400 ppm |
| ORP         |       +94 mV |
| H₂ ppb      |            0 |

Compared with recent non-menstrual fresh plasma, this sample read substantially higher in conductivity and TDS, with a notably acidic pH.

---

## Run 2 — First Distillation (Blow-Over Batch)

This distillation experienced blow-over and was set aside for redistillation.

| Parameter   |     Reading |
| :---------- | ----------: |
| Temperature |      36.6°C |
| pH          |         9.1 |
| EC          | 2,490 µS/cm |
| TDS         |   1,280 ppm |
| ORP         |      -41 mV |
| H₂ ppb      |          84 |

### Change Through Distillation

| Parameter  |    Raw | Distilled |
| :--------- | -----: | --------: |
| pH         |    5.1 |       9.1 |
| EC µS/cm   | 10,690 |     2,490 |
| TDS ppm    |  5,400 |     1,280 |
| ORP mV     |    +94 |       -41 |
| H₂ ppb     |      0 |        84 |

pH shifted from acidic to strongly alkaline. EC and TDS each dropped by roughly three-quarters. ORP inverted from positive to negative. Detectable dissolved hydrogen appeared in the distillate where none was present in the raw source.

## Observations

**The distillation signature held, from an unusually concentrated start.** Despite a raw source more than twice as concentrated as recent fresh plasma, the run produced the same arc seen across the June sessions: acidic-to-alkaline pH, falling EC and TDS, positive-to-negative ORP, dissolved hydrogen emerging only after distillation.

**Conductivity and TDS fell with each pass.** 10,690 → 2,490 → 915 µS/cm; 5,400 → 1,280 → 461 ppm. The redistillation brought the blow-over batch down into the same clean range the other successful redistillations have reached.

**pH rose across both passes.** 5.1 raw → 9.1 first distillation → 9.7 second. The alkaline volatile fraction remained the stable anchor.

**ORP inverted and held.** +94 mV raw → -41 mV → -42 mV. The charge inversion is consistent with every distilled fraction in the record.

**Dissolved hydrogen emerged after distillation and persisted.** 0 ppb raw → 84 ppb → 88 ppb, holding through redistillation rather than falling with EC and TDS as it did in the June 10 redistillation.

**The standout is the raw morning reading.** EC 10,690 µS/cm and TDS 5,400 ppm — far above the recent non-menstrual fresh plasma values.

---

## What Remains Open

Whether the elevated raw concentration reflects the menstrual blood (artava) contribution, morning concentration, hydration status, or some combination cannot be determined from these readings alone. It is recorded here as a clear, standout signal in the dataset and a question for the seasonal and cyclical record to revisit — the first measured entry of the menstrual fraction into the distillation work.

The hydrogen behaviour is a second open thread: it rose into the distillate and held across redistillation, where the June 10 redistillation showed H₂ falling alongside EC and TDS. Whether that difference is a property of menstrual-fraction material or of these specific conditions is not established.
