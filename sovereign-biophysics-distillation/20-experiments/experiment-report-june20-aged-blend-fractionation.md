# Experiment Report: June 20 — Aged Blend Fractionation

**Operator:** Ana Seahawk
**Date:** June 20, 2026
**Input:** Minimum one-month-aged blended plasma from Ana and partner
**Run structure:** Two independent first-pass distillations from the same aged source bottle; Run B repeated the operation to capture the serum fraction

---

## Boundary Note

This is a blended-source household record.

The measurements belong to the behavior of one aged bottle under distillation, not to one person's physiology. The value of the run is fractionation repeatability: what happened when two portions from the same aged source were distilled separately under the same household conditions.

No medical or therapeutic claim is made.

---

## Run Context

The source material was at least one month old. It was already alkaline, conductive, and near-neutral by ORP before distillation.

Run A began as an ordinary fractionation run. The still was not stopped early enough to retain a separately measured serum fraction; by the time the state was noticed, the apparatus was close to shutdown and little remained in the chamber.

Run B used a fresh portion from the same aged bottle. It was not a second distillation of Run A. It was performed so the serum could be isolated and measured as its own fraction.

That accident made the session more useful. The archive gained two independent spirit and essence readings from the same aged source, plus one serum profile from the repeated run.

## Cross-Run Comparison

| Fraction | Run A | Run B | Pattern |
| :--- | :--- | :--- | :--- |
| Spirit pH | 10.0 | 9.9 | nearly identical |
| Spirit EC | 16.83 mS/cm | 15.07 mS/cm | same high-conductivity band |
| Spirit ORP | -94 mV | -97 mV | nearly identical |
| Spirit H₂ | 190 ppb | 198 ppb | nearly identical |
| Essence pH | 9.3 | 9.7 | same alkaline band |
| Essence EC | 1368 µS/cm | 649 µS/cm | lower than spirit in both runs |
| Essence ORP | -58 mV | -58 mV | identical |
| Essence H₂ | 124 ppb | 118 ppb | close range |

The spirit cuts are the strongest repeatability signal in this session. Across two separate first-pass runs, pH, ORP, conductivity, and dissolved hydrogen all stayed close.

The essence cuts also clustered, especially in ORP and H₂. Conductivity moved more widely, but remained far below the spirit fractions.

The serum, measured only in Run B, occupied a different profile: lower pH, higher conductivity, positive ORP, and no measurable H₂.

---

## Observations

**The aged source was already alkaline and conductive.** The raw blend entered the still at pH 9.1 and 23.0 mS/cm, with 11.5 ppt salinity and no measurable dissolved hydrogen.

**The spirit fraction repeated closely across independent runs.** Run A spirit and Run B spirit both landed near pH 10, strongly negative ORP, high conductivity, and approximately 0.2 ppm H₂.

**The essence fraction retained the same ORP in both runs.** Both essence readings measured -58 mV. H₂ also stayed close, while EC differed by roughly half.

**The serum separated as the conductive, oxidizing pole.** The Run B serum measured pH 6.1, 26.0 mS/cm, +45 mV, and 0 ppb H₂. In this session, serum carried the densest conductive reading and the only clearly positive ORP after fractionation.

**The accident improved the record.** Because Run A missed the serum capture and Run B repeated the run from the same aged bottle, the archive gained a repeatability comparison it would not otherwise have had.

---

## What Remains Open

Only one serum fraction was measured. The archive cannot yet say whether this serum profile repeats across aged batches, different bottles, different fill levels, or different stopping points.

The two essence conductivity readings also need more context. ORP and H₂ held closely, while EC varied from 1368 to 649 µS/cm. Collection timing, fraction boundaries, chamber level, cooling state, and measurement timing may all matter.

The blended source is a boundary condition. This run can support apparatus and fractionation questions, but it cannot be treated as a single-subject physiological finding.

---

## Working Hypothesis

Aged blended material may produce a repeatable high-alkaline spirit profile across separate first-pass runs from the same bottle. Serum may occupy the opposite pole within the same fractionation sequence: dense, conductive, positive-ORP, and without measurable H₂.

The spirit repeated. The serum marked the edge.
